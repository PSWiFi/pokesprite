PokéSprite

Pokémon icon database and sprite sheet generator

This is a database containing icons for all the Pokémon in the National Pokédex, along with the icons for every single in-game item. It also includes shiny versions. Icons can be used either as individual files, or through an optimized sprite sheet (CSS and JS).
